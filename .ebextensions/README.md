# so i dont forget

important/sensitive environment variables for production are to be set in the eb cli thing. they are:

- `PORT`
- `NODE_ENV`
- `SENDER_ADDRESS`

### only for production:

- `MAILGUN_API_KEY`
- `MAILGUN_DOMAIN`

### only for testing:

- `ETHEREAL_MAIL_USERNAME`
- `ETHEREAL_MAIL_PASSWORD`
